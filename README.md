# SiboWeb

SiboWeb is a ticket management system that enables efficient ticket administration processes for events such as concerts, games, and more.
<br>
<br>
<b>This is the Web version</b> of the application developed in Java to be used in Internet browsers.
<br>
<br>
JSP2 and GlassFish Server were used

Features:
- Authentication mechanism via MySQL database
- Admin/User selected views depending on user roles
- View Tickets
- Buy Tickets
- Create venues, concerts, categories, etc
- add Selling stores, etc.
